CodeAlpha Stock Portfolio Tracker

Description

A simple Python program that calculates the total investment value of a stock portfolio based on user input and predefined stock prices.

Features

- Accepts stock name from the user
- Accepts quantity of shares
- Calculates total investment value
- Displays an error message for invalid stock names

Technologies Used

- Python
- VS Code

Sample Stocks

- AAPL
- TSLA
- GOOG
